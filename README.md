# BharatEV Franchise Operations & ROI Intelligence Dashboard

An AI-assisted operational intelligence platform designed for EV franchise management, executive monitoring, and operational decision-making.

This project was built as a prototype focused on:

* franchise operations visibility
* executive-level dashboarding
* AI-assisted operational insights
* operational bottleneck monitoring
* role-based business intelligence
* realistic startup execution workflows

---

## Overview

The dashboard simulates a real-world EV franchise operations platform actively used by:

* CEO
* CTO
* Co-Founder
* Business Heads
* Operations Teams
* Regional Operations Managers

The focus was to create a platform that feels:

* operationally realistic
* startup-oriented
* execution-focused
* AI-assisted but human-controlled
* enterprise SaaS-grade
